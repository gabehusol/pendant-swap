# samples/

Place your test images here. This directory is gitignored to prevent accidental
commit of model photos or product images.

Expected files for the end-to-end test:
- `model.jpg`    — the model/lifestyle photo
- `pendant.jpg`  — the pendant product photo (ideally on white or solid background)
- `reference.jpg` — a known-good hand-built composite to validate QA against

None of these files are committed to the repository.
